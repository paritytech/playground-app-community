# Contributing

This is experimental, proof-of-concept code developed and published by
Parity as a reference. **External pull requests are not accepted at this
time.**

You are welcome to:

- **Fork the repository** and modify it for your own use — see the
  [LICENSE](LICENSE) for terms (GPL-3.0-or-later).
- **Open an issue** to report a bug or share feedback. We may or may
  not respond, depending on bandwidth.
- **Report security vulnerabilities** by email to
  **security@parity.io** — see the [paritytech org SECURITY
  policy](https://github.com/paritytech/.github/blob/main/SECURITY.md)
  and the [Security section in the README](README.md#security-status)
  for scope and expectations.

If this policy changes (e.g. we open the repo to community
contributions), this file will be updated.

## Code of conduct

This repository follows the
[Parity organisation Code of Conduct](https://github.com/paritytech/.github/blob/main/CODE_OF_CONDUCT.md),
inherited by default from `paritytech/.github`. Treat other participants
with respect in any issue threads, fork discussions, or other public
interactions related to this codebase.
